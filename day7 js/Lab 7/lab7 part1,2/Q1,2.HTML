Selectors

Selectors are used to fetch page elements inside an Array

  • tag: Selects all elements of the <tag> type
  • .className: Chooses all elements with the className class applied
  • #identifier: Picks the element with the id of identifier
  • tag.className: Selects all <tag> elements with the className class applied

Css Hierarchy Selectors

To select elements based on their position in the document tree, use the Hierarchy Selectors.

The Hierarchy Selectors work by examining the position of target elements relative to other elements:

  1. parent > child: Selects "child" elements that are direct descendants of the "parent"
  2. ancestor descendant: Picks "descendant" elements as long as they have an "ancestor" element somewhere above them
  3. prev + next: Selects the "next" element if it is immediately preceded by a "prev" element
  4. prev ~ siblings: Chooses all "siblings" elements that come after a "prev" element
